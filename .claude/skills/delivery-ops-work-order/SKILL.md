# Delivery Ops Work Order

Read `docs/agent-skills/delivery-ops-work-order.md` before generating or modifying evidence packets or work orders.

Key constraints:

- Build evidence before work orders.
- Keep Bug Fix and Feature templates separate.
- Include allowed scope, forbidden changes, acceptance criteria, and risk level.
- Mark conflicts and unknowns explicitly.
- Do not ask executors to perform unrelated refactors.
